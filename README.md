### -# By Sayanide.

A beautiful fork of the **Midnight Vencord** theme with fresh frosted glass overlays and enhanced customization.

---

### ✨ What's Different?

The original Midnight theme didn't include a ready-made frosted translucent effect.  
This version adds a clean, modern **frosted glass** aesthetic while keeping the beautiful dark Midnight vibe.

> If you want the original Midnight theme, you can find it here:  
> **[Midnight Theme →](https://betterdiscord.app/themes/midnight)**

---

## 🎨 How to Change the Background Image

1. Open the CSS file in any text editor.
2. Scroll down until you find this section:

```
/* ================================================= */
/* =============== BACKGROUND IMAGE ================= */
/* ================================================= */
```

3. Edit the background URL here:

```
/* Edit the URL from this line */
--background-image-url: url('https://files.catbox.moe/r117dk.png');
```

---

**URL** for images will be available below that area in the file.

---

**How to apply:**
- Copy the URL of your chosen background
- Paste it in the `--background-image-url` line
- Save the file (`Ctrl + S`)

Enjoy your fresh new frosted look! ✨
